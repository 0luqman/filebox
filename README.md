# Filebox

**Filebox** is a modern, block-based workspace application that replicates the core experience of Notion. It is built to organize your thoughts, projects, and information in a flexible, hierarchical structure.

## Features

*   **Block-Based Editing**: Every piece of content (text, heading, list, image) is a block that can be transformed and moved.
*   **Slash Commands**: Type `/` to instantly access the command menu and insert new blocks.
*   **Nested Pages**: Create infinite depth of pages within pages to structure your knowledge.
*   **Dark Mode**: Built-in support for light and dark themes.
*   **Rich Media**: Support for images, code blocks, quotes, and more.

## Tech Stack

*   **React**: UI Library
*   **TypeScript**: Type safety and developer experience.
*   **Tailwind CSS**: Utility-first styling.
*   **Lucide React**: Beautiful icons.

## Getting Started

1.  Clone the repository.
2.  Install dependencies (if applicable to your environment).
3.  Run the development server.
